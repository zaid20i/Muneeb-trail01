# Car Rental Backend

## Setup

1. `cd backend`
2. `npm install`
3. `node src/utils/seed.js` (to seed demo data)
4. `node src/app.js`

## Environment
- Set variables in `.env` (see example)

## API
- All endpoints are under `/api/`
- Auth: `/api/auth`
- Drivers: `/api/drivers`
- Vehicles: `/api/vehicles`
- Contracts: `/api/contracts`
- Payments: `/api/payments`
- Fines: `/api/fines`
- Notifications: `/api/notifications`
- Support: `/api/support`
- Documents: `/api/documents`
- Reports: `/api/reports`
- Settings: `/api/settings`

## File Uploads
- Uploaded files are stored in `src/uploads/`
